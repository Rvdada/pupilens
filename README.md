# Landing Page Pupilens - Identidade visual ajustada

Projeto estático pronto para Vercel.

## Como atualizar
1. Substitua o `index.html` do repositório pelo arquivo desta pasta.
2. Faça commit e push na branch `main`.
3. A Vercel deve iniciar um novo deploy automaticamente.

## Observações
- As imagens do produto usam URLs oficiais do site atual da Pupilens.
- Os CTAs apontam para `https://demo.pupilens.com.br/demonstrativo-de-lentes`.
- Caso queira rastrear conversões, instale GTM/Meta Pixel no `<head>` do `index.html`.
